## Pre-requirements
You will need Python 3.8 and Flask-WTF 0.14 to build the application, find more informations on 
[the Python documentation](https://docs.python.org/3/) and on [the Flask-WTF documentation](https://flask-wtf.readthedocs.io/en/stable/). 

## Contributing

Please note we have a coding conventions, in order to sanitize coding standards,
please follow [this style guide](https://www.python.org/dev/peps/pep-0008/).

To contribute :
```bash
# Clone the application
git clone https://github.com/Florian-lg/dianorian.git
cd dianorian

# Build and initialize the project
flask run
```
